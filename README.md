# 🏥 AI Medical Assistant (Multi-Agent System)

An AI-powered healthcare assistant built using a **multi-agent architecture**.  
The system helps users obtain medical information through conversational AI and medical image analysis.

It integrates **LLMs, retrieval systems, and computer vision models** to assist users with health-related queries.

---

## 🚀 Features

### 💬 Medical Chat
Users can ask medical or health-related questions and receive responses generated by large language models.

### 📚 Knowledge Retrieval (RAG)
Retrieves relevant information from stored medical documents to improve response accuracy.

### 🌐 Web Search Integration
Fetches up-to-date medical information from online sources.

### 🖼️ Medical Image Analysis
Users can upload medical images such as X-rays or scans for AI-based analysis.

### 🛡️ Safety Guardrails
Includes built-in safety checks to prevent harmful or misleading responses.

### 💾 Session Memory
Maintains conversation history to provide context-aware responses.

---

## 🧠 Multi-Agent Architecture

The system routes queries to specialized agents.

- **Router Agent** – decides which agent should process the request  
- **Medical Chat Agent** – handles general medical conversations  
- **RAG Agent** – retrieves data from the medical document database  
- **Web Search Agent** – gathers latest medical research information  
- **Vision Agent** – analyzes uploaded medical images  
- **Guardrails Agent** – ensures safe and appropriate responses

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/p
SruthiReddy7/Medical-assistant.git
cd Medical-assistant
```

---

### Backend Setup

```bash
cd backend
pip install -r requirements.txt
```

Create environment file:

```bash
cp .env.example .env
```

Add API keys:

```
GROQ_API_KEY=your_api_key
TAVILY_API_KEY=your_api_key
```

---

### Frontend Setup

```bash
cd frontend
npm install
```

---

## ▶️ Running the Application

### Start Backend

```bash
cd backend
uvicorn main:app --reload
```

### Start Frontend

```bash
cd frontend
npm start
```
